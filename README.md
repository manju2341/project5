# Project 9: Simple Counter

A simple counter application that demonstrates basic JavaScript functions and DOM manipulation.

## Concepts Covered:
- JavaScript Functions
- DOM Manipulation (getting elements, updating content)
- Event Listeners
- Variables and State Management

## Features:
- Increase counter
- Decrease counter
- Reset counter
- Color changes based on count value (green for positive, red for negative)

## How to Use:
1. Open `index.html` in a web browser
2. Click the buttons to increase, decrease, or reset the counter
3. Watch the number and color change!
